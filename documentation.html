Documentation

Customize NovaEdge in minutes.

A complete guide to personalizing your template, from colors to copy.

How to Change Colors

All colors are defined as CSS variables at the top of style.css. Look for the :root section and modify --primary, --accent, and --bg to match your brand.

For dark mode, update the body.dark-mode section with your color values. Changes apply instantly across the entire template.

How to Change Fonts

NovaEdge uses "Sora" for headings and "Manrope" for body text. To use different fonts, edit the @import URL in the <head> section of each HTML file, then update the font-family values in style.css.

Alternatively, use Google Fonts, Adobe Fonts, or any web font service that supports @import.

How to Replace Images

All template images are in the /images/ folder as SVG files. Replace them with your own images (JPG, PNG, or WebP) by uploading to the same folder and updating the src attributes in the HTML files.

For best performance, optimize images before uploading (use tools like TinyPNG or ImageOptim).

How to Enable Dark Mode Persistence

Dark mode is already enabled in NovaEdge! The template automatically saves your preference to localStorage. Simply click the moon/sun icon in the header to toggle between modes.

The setting persists across page refreshes and browser sessions—no additional setup required.

How to Set Up Form Submissions

The contact form on this template validates inputs on the front-end. To process submissions, integrate a backend service like Formspree, Netlify Forms, or EmailJS. Update the id="contactForm" form's action attribute to point to your service.

The existing JavaScript validation will still work—ensure field names match your backend requirements.
