Frequently Asked

Got questions? We have answers.

What's included in the template? +

All packages include HTML, CSS, JavaScript, documentation, and SVG assets. Premium plans include a launch consultation and priority email support.

Can I use this for client projects? +

Yes, both personal and commercial use are permitted. You can use NovaEdge for client projects without additional licensing fees.

How do I customize the colors and fonts? +

All colors and fonts are easily customizable through CSS variables at the top of style.css. See the documentation page for a step-by-step guide.

Is the template mobile responsive? +

Yes, NovaEdge is fully responsive and optimized for desktop, tablet, and mobile devices. All features work seamlessly across screen sizes.
