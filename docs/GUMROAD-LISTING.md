## Short Description
A modern, responsive, and conversion-focused HTML template for agency, startup, and business websites. Built with clean HTML, CSS, and JavaScript, with no framework dependency.

- about.html

- Responsive layout (desktop, tablet, mobile)
- Sticky navbar with dropdown
- Dark mode toggle with saved preference
- Smooth scroll and reveal animations
- Portfolio cards with polished overlays
- Pricing section with hover interactions
- Contact form UI with validation
- Back-to-top button
- Premium-style footer layout
